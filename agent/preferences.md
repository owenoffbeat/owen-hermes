# Owen의 작업 스타일 & 선호도

last updated: 2025-04-17

## 커뮤니케이션

- 한국어로 대화
- 핵심을 먼저, 설명은 그 다음
- 불필요한 확인 질문 최소화 — 판단 가능한 건 먼저 실행

## 기술 스택 선호

- Backend: Python / FastAPI
- DB: PostgreSQL (시계열은 TimescaleDB)
- Frontend: React + Recharts/ApexCharts
- 스케줄러: APScheduler

## 작업 방식

- 큰 작업은 단계별로 쪼개서 진행
- 중요한 변경 전에 반드시 git commit (되돌릴 수 있게)
- 오류 발생 시 원인 파악 후 수정 — 무작정 재시도 금지
